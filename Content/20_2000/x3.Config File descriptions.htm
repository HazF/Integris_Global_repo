Configuration Files

Integris configuration files define the parameters for Radio Head testing. These parameters include the types of tests transmitted to the RRH as well as pass/fail values.

The table below describes the configuration files included with the Integris 2000 system.

File Name Description
Default-LTE 'Standard' CPRI LTE. No inventory polling but does check for a loop-back condition on the cable.
Ericsson-LTE-Basic Contains test parameters for Ericsson configuration LTE
Ericsson-LTE Contains test parameters for Ericsson configuration LTE, plus attempts to read BBU and / or SFP inventory. Includes loop-back detection.
Lucent-LTE-Basic Contains test parameters for Alcatel-Lucent configuration LTE
Lucent-LTE Contains test parameters for Alcatel-Lucent configuration LTE, plus attempts to read BBU and / or SFP inventory
Selftest Used with a loopback connector at the end of a cable attached to Integris test port 1, to check the Integris 2000 system is working correctly
