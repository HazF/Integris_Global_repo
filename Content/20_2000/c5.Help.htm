Accessing Integris Help

Location  
GUI or Keyboard Click on the F1 key at any time
Classic menu System > Help
Modern menu Help > View Help

Integris 2000 comes with a Help system to explain the setup and use of the test system.

To access Help:

  • Click on the F1 key at any time
  • On the Classic menu system, select the System menu and click on Help
  • On the Modern menu system, select the Help menu and click on View Help
