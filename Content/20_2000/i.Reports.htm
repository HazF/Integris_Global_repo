At the end of each test, Integris 2000 creates a set of result values that were accumulated during the test. As detailed in the procedures for the individual tests, these can be saved and accumulated into reports.

  • Click on the REPORT menu to view options related to report preparation.

Modern menu

Classic menu

View Hot key, Ctrl + M: (Modern menu only). Opens the current report and shows any test results that have been included so far.

Select: Allows you to open a previously saved report and any test results it may contain. Results from any new tests will be added to this report.

Clicking on any of the test results in the opened report expands its contents for viewing.

Clicking on the yellow Add to Report button after running any of the tests will add the test to this report. In this way you can create report files with a list of individual results.

New: Creates a new empty report using the current date and time to form the filename.

Clicking on the yellow Add to Report button after running any of the tests will add the test to this report. In this way you can create report files with a list of individual results.

Edit: Allows you to edit the file name and add a site name along with any notes.

Clear: Clears all test results, site name and notes from the current report and sets the date field to the current date.

Export PDF: Opens an explorer window to select a location for the exported Portable Document Format (PDF) file, then opens the report for viewing. (See Figure 1.1. )
