Installation Test Sequence

The Integris 2000 Radio test checks the operation of the whole RAN installation and successful completion of this test signals a Pass for the site. This section describes a complete RAN installation test sequence, then records the results to a report file. This file can be exported in Portable Document Format (PDF) for record keeping purposes.

Radio Head testing consists of a sequence of individual tests that initialize the CPRI link and Remote Radio Head (RRH). They then verify that the link is capable of reliably supporting CPRI traffic at the current speed.

Connecting Integris 2000 to an RRH

To test or monitor a radio head installation connect the cable assembly to Integris test port 1.

  1. Remove cable connecting the RRH to the BBU, by disconnecting it from the SFP in the BBU
  2. Remove the SFP from the BBU port where the cable to the RRH was previously connected
  3. Place this SFP in port 1 of the Integris System

  4. Connect the cable from the RRH to the SFP in port 1 of the Integris System (See Fig 1.2 ).

  5. Power on the Radio Head.

Integris 2000 Test Port 1

Before testing you must set Integris 2000 to the correct link rate for the test.

Setting the link rate:

  1. Click on the Rate icon in the Status Panel at the top left of the main window. A drop down menu will appear. The currently selected rate is highlighted
  2. Click on the required link rate.

The text on the icon will display the words "Active Link" in green if the SFP recognizes an active CPRI link, or a red error message if a problem exists.

Radio Head testing is accomplished by transmitting a test sequence that is matched to the equipment being tested.

These tests are defined by a configuration file, which must be loaded before any testing can take place.

Loading a configuration file:

  1. Click on the TEST CONFIG icon at top center of the main screen

  2. Select the required configuration file from the list.

Report files contain the results of individual RRH, Cable and SFP tests. The first step in a test sequence therefore is to create a report file to contain them. The report file will contain one or more sets of results from the tests you run.

Creating a test report:

  1. Select the REPORT menu and click NEW. A new blank report will be created with a default format file name and the current date.

  2. On the Report menu, click on Edit. In the report window that opens add site details and any notes you might want to add. You can also edit the file name to something more appropriate.

Editing report details
Note: You can go back and edit any of these fields at any time by clicking on REPORTS > EDIT.

Now that the report has been created and details edited you can run the installation test sequence.

Running the Tests:

Now that the report has been created to contain the test results, we can now run the Installation test.

  1. On the Test menu, click Select Test, then Radio.
  2. Start the test by clicking on Test > Run (or pressing F5)
  3. When the test has completed a yellow message box will open offering to save the results to the currently open report file. Click on OK

Processing the report file

To see the report containing the results of individual tests:

  1. Go to the Report menu and click View.
  2. The screen will now display the report containing a list of individual test results that can be expanded or collapsed using the arrows to the left of the result name.

    Report showing test results

Exporting the report file in Portable Document Format (PDF)

  1. Click on the Report > Export PDF. An Explorer window will open allowing you to name and save the PDF file to a location of your choice.

  2. Click SAVE.
