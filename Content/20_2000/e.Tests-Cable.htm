Testing a Cable

Location  
GUI or Keyboard F5 to run test
Classic menu Test > Select Test > Cable
Modern menu Test > Select Test > Cable

Integris 2000 can run a sequence of tests to check the integrity of the cable and connectors connected to the test port. Tests include those for continuity, cable length and any losses in the cable.

NOTE:This test requires a loop-back, or a device configured in loop-back mode, at the far end of the cable.

To run the cable tests:

  1. Open the TEST menu and click on SELECT TEST, then CABLE.
  2. Click on RUN from the same menu (or press the F5 key),
