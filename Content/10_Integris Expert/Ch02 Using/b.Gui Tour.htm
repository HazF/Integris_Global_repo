The Integris Expert GUI

When you have started Integris Expert the Expert Graphical User Interface (GUI) appears. The GUI consists of a tabbed pane to the left, and a link status pane with SFP status and vital product data to the right. (See Fig 1.1 )

The Integris Expert GUI

Control Description

1. C&M Capture Tab

 This tab displays controls for setting the link speed, and enabling capture of Fast and Slow C&M data streams.
2. Raw CPRI Frame Capture Tab This tab displays controls for setting the link speed and capturing raw CPRI frame data to a file.
3. CPRI  Link Speed dropdown Sets the CPRI link speed.
4. CPRI Channel dropdown Selects either Fast C&M or Slow C&M data channels.
5. C&M Capture button When you have set the correct link speed and data channel, clicking on the Capture button opens a Wireshark window and displays live C&M data that has been extracted from the CPRI link.
6. Ports 0 & 1 Status panes Displays information on the link & SFP status, plus SFP metrics and product data.
7. Raw Data file size & Frame capture bytes Sets the raw data file size. When the captured file reaches the set size, capture will stop. Capture(Bytes) sets the number of bytes in each frame that will be captured. For example, if this value is set at 16, then the first 16 Bytes in each frame will be stored in the output file. Frame size increases with link speed.
8. Output File name Sets the name and path of the output file.
9. Raw Capture controls Buttons to start, stop capture, or view captured files.
