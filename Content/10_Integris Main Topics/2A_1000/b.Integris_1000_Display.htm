Integris 1000 Display

The Integris 3000 display consists of five main areas (See Figure 1. 1: )

  1. System Status This displays quick reference values showing how key parameters have been set. If System > Source is set to Live it additionally shows the link status (Options: Red “Down”, Green “OK”). The System Status area is 'live' so values can be set by clicking on the number and setting a value using a keypad or slider, or selecting the value from a list.

    2. The Integris 3000 Screen
  2. Snapshot: Take and display screen shots
  3. Menu Selector: Displayed when Classic View is selected from System > Preferences, This selector is located at the bottom of the screen and allows the different Classic menus to be selected.
  4. Power Spectrum: A traditional “Bart’s Head” displaying amplitude against frequency. Combinations of Power, Waterfall and Persistent graphs can be displayed using Graph > Type
  5. Crosshairs: These can be selected or deselected using ‘Graph > Crosshair. The values at this location are shown at 6.
  6. Crosshair Values: The are the values at the crosshair intersection (5)
  7. Menus: Seven menus are available and are selected using the Menu Selector (3)
