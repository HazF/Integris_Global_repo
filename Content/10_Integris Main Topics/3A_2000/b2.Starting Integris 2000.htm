Starting Integris 2000

Start the Integris 2000 application by double-clicking on the Integris 2000 shortcut on the Windows desktop.

The Integris 2000 splash screen will display briefly before the Graphical User Interface (GUI) opens.

NOTE: The Integris 2000 application 'remembers' the last test run on the system and will display this when the GUI opens.
