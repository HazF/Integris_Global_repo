Loading a Configuration File

Location    
GUI or Keyboard Configurationicon at the top of the Main Window, or Auto Detect
Classic menu Test > LoadConfig
Modern menu Test > Load Config

Radio Head testing is accomplished by transmitting a test sequence that is matched to the equipment being tested. These tests are defined by a configuration file.

To load a configuration file, click on the Test Config icon at top center of the main screen. Select the required configuration file from the list. (See Fig 1.1 )

See "Configuration Files", page 1 for more information on configuration files

Click here for more information on Configuration Files

NOTE: From time to time you may need to update or add Configuration files. These are located in the C:\Integris-2000\Configs\ folder.
