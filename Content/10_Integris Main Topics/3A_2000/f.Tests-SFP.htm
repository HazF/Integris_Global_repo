Testing an SFP

Location  
GUI or Keyboard F5 to run test
Classic menu Test > Select Test > SFP
Modern menu Test > Select Test > SFP

NOTE: These SFP tests relate to an SFP fitted into Integris test port 1. They should not be confused with the SFP information reported by the Radio test. The SFP information returned during the Radio test relates to the SFP connected inside the RRH.

The Integris 2000 is able to read the vital product data (VPD) from an SFP located in its test port. It can also provide basic functional testing, as well as providing metrics such as operating temperature and transmit & receive signal power.

  1. Open the TEST menu and select SELECT TEST, then click on the SFP option.
  2. Now click on RUN from the same menu (or press the F5 key),

SFP Results

When the test sequence has been completed you can optionally save the test results by adding them to a report. To add to an existing or new report file, click on the yellow Add To Report button.(See 'Working with Test Result Files').

You can now choose a name for the test results, or you can accept the default name based on the date and time of the test. Once you have finished editing the results name, click on OK.
