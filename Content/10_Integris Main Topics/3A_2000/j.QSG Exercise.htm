Running a Sequence of Tests

This introduction to Integris 2000 will follow a complete test sequence, then record the results to a file. It will then describe how to convert it to Portable Document Format (PDF) for record keeping purposes.

To create and run a test sequence:

  1. Open the REPORT menu and click on NEW. A new blank report will be created.

  2. On the REPORT menu, click on EDIT. A window will open with a pre-formatted file name (based on the date and time of the test), with fields for the site name and any notes.

    Editing the Report Details

    Note: You can go back and edit any of these fields at any time by clicking on REPORTS > EDIT.

    SFP Test

  3. The first test will check out the SFP. Open the TEST menu and click on SELECT TEST, then SFP.
  4. To run the test go to the TEST menu, click on RUN (or press F5)

  5. When the test has completed a yellow message box will open offering to save the results to the currently open report file. Once you have edited the name and details for this set of results, click on OK

    Cable Test

  6. Now we will test the cable. In order to test the cable we will need a loop-back at the far end. Either fit a physical loop-back, or configure a device connected to the far end of the cable for loop-back.
  7. Open the TEST menu and click on SELECT TEST, then Cable.
  8. Run the test by going to the TEST menu and clicking on RUN (or press F5)
  9. When the test has completed a yellow message box will open offering to save the results to the currently open results file. Once you have edited the name and details for this set of results, click on OK

    Radio Head Test

  10. Next is the radio head itself. Connect a radio head to the far end of the cable and power it on.
  11. Open the TEST menu and click on SELECT TEST, then RADIO.
  12. Run the test by going to the TEST menu and clicking on RUN (or press F5)
  13. When the test has completed a yellow message box will open offering to save the results to the currently open results file. Once you have edited the name and details for this set of results, click on OK

    Processing the report file

  14. To see the report containing the results of individual tests, Click on the REPORT menu, then VIEW. The screen should now be displaying a list of test results that can be expanded or collapsed using the arrows to the left of the result name.

    Test Sequence Results

    Now we will export the file in Portable Document Format (PDF).

  15. Click on the REPORT menu and then Export PDF. An Explorer window will open to allow you to save the PDF file to a location of your choice. When you are ready, click SAVE.
