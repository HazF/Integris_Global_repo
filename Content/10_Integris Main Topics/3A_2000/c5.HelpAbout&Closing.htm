Accessing Integris Help

Location  
GUI or Keyboard Click on the F1 key at any time
Classic menu System > Help
Modern menu Help > View Help

Integris 2000 comes with a Help system to explain the setup and use of the test system.

To access Help:

  • click on the F1 key at any time
  • On the Classic menu system, select the System menu and click Help
  • On the Modern menu system, select the Help menu and click View Help

Closing the Integris 2000 Application

Closing the Integris 2000 Application

Location  
GUI or Keyboard

Click on white X at top right of the Integris 2000 window or press Alt+F4 keys
Classic menu System > Exit
Modern menu System > Exit

You can close the Integris 2000 application by either clicking the white X at top right of the Integris main window, or opening the System menu and clicking on Exit. The Integris system will return to the Windows desktop.
