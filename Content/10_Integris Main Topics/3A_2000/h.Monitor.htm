Location

Classic menu Test > Monitor

Integris 2000 tests are set to run for a period of time, then conclude with a Pass or Fail result.

Radio head monitoring is a long term test designed to exercise the radio head over a period of time, usually to facilitate crews to adjust or modify the installation.

Note: it is also possible to run monitoring tests to the SFP or cable, though monitoring is more common on radio heads.

To monitor radio head operation:

  1. Select the TEST menu, and click on SELECT TEST. Choose the RRH option.
  2. Click on the MONITOR menu option (Optionally press CTRL + M). If using Classic menus, the text under the MONITOR button changes to a green ON.

  3. Click RUN (Optionally press F5).

    The word "Monitoring" will appear in yellow at the top right of the window to indicate status. The test results pane shows the current values for the test.

  4. To stop monitoring, select the Test menu and click on STOP.

Note: Unlike the other tests, monitoring will continue until you click on the STOP button or disable monitoring. If you stop then restart monitoring, all values will be reset.
