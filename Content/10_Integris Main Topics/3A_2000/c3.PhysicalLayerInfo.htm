You can access information on the link status by clicking on the Link icon on the main window. This information relates to the SFP and link connected to the default Integris 2000 test port (port 1).

Note: Do not confuse SFP sometimes referred to in link status reports with the SFP information in radio tests.

  • The SFP referred to in some link state error messages relates to the SFP fitted into Integris test port 1
  • The Integris 2000 radio tests report information related to the SFP fitted to the RRH

 

Information:all SFP vendors provide machine readable information regarding the SFP's capabilities. This information is called the Vital Product Data (VPD) and includes the SFP manufacturer, supported link rates, optical wavelength, etc. 

The Integris System can read this information and present it to the user so they can verify that the SFP is suitable for the application in which it is being used.
