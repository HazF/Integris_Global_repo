NOTE: Inventory polling applies to selected manufacturers' configurations only. Contact your dealer for the latest configuration files.

The Integris 2000 RADIO tests include enhanced inventory listing for some manufacturers' equipment. This can be useful for fault finding and verifying software and firmware levels.

In this group of tests, the results are expanded to include Radio Head inventory information, including Manufacturer, model and serial number, as well as firmware level and service data.

NOTE: The SFP results section below the Inventory results lists information related to the SFP fitted to the Radio Head. These results should NOT be confused with those gathered during the Integris SFP Tests described elsewhere. See "Testing an SFP", page 1.

To run Radio tests BBU emulation tests:

  1. At the top of the screen click on the yellow Test Config button and select a configuration file that supports the expanded BBU testing .
  2. Open the TEST menu and click on SELECT TEST. Click on the RADIO option.
  3. Now click on RUN from the Test menu (or press the F5 key),

Manufacturers' Configuration Files

BBU Emulation Test Result Screen

Once the test sequence has been completed you can optionally save the test results by adding them to a report. To add to an existing or new report file, click on the yellow Add To Report button (See "Working with Test Reports", page 1).

You can now choose a name for these results, or you can accept the default name based on the date and time of the test. Once you have finished editing the name, click on OK.
