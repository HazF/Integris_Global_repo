Troubleshooting a RAN Installation

If a radio head fails the initial Integris 2000 Radio tests then there is a logical debug sequence to follow to locate the component giving the problem. This section suggests a sequence of actions and details the required tests.

Repeat the radio test at Ground Level

Bringing the Radio Head back to ground level makes replacing components, and unit configuration easier.

Connecting Integris 2000 to an RRH

To test a radio head installation connect the cable assembly to Integris test port 1. (See Fig 1.1 ).

  1. Take a full duplex cable and connect one end to the RRH

  2. Connect the other end of the cable to the SFP in port 1 of the Integris System (See Fig 1.2 )(See Integris 2000 Test Port 1 ).

    NOTE: you MUST install exactly the same SFPs into the Integris test port as used in RRH & BBU. If there is any doubt as to the SFP's specification, remove the SFP from the BBU and use that in Integris test port 1

  3. Repeat the Integris Radio test sequence by Openning the TEST menu and selecting SELECT TEST. Then click on the RADIO option.
  4. Now click on RUN from the same menu (or press the F5 key),

    (See "Testing a Remote Radio Head", page 1)(See also Testing a Remote Radio Head)

Test local and remote SFPs

If the radio head fails the Radio test, the next option is to test the SFPs in the Integris test port 1 and then the one in the Radio Head itself.

With the current SFP still in Integris test port 1:

  1. Fit a loopback cable to the SFP being tested (See Fig 1.4 ).
  2. Open the TEST menu and select SELECT TEST, then click on the SFP option.
  3. Now click on RUN from the same menu (or press the F5 key),

Loopback Connection

If this (local) SFP passes the SFP test, remove the SFP in the radio head and insert that in Integris test port 1 (See Fig 1.3 ). Fit the loopback cable and repeat the test.

Testing the Cable

The next logical step is a test of the cable linking the Integris system to the Radio Head. This procedure assumes the SFP currently in Integris test port 1 has passed the SFP test.

  1. Insert one end of the cable into Integris test port 1
  2. Fit a looback to the far end of the cable (See Fig 1.4 )
    • An alternative if no loopback connector is available is to remove one of the optical fibers from the end clip that makes up the duplex cable, and insert opposite ends of the same fiber into the TX and RX connectors of Integris test port 1.
  3. Open the TEST menu and select SELECT TEST, then click on the SFP option.
  4. Now click on RUN from the same menu (or press the F5 key),

(See "Testing a Cable", page 1) (See also Testing a Cable)

Monitor Mode Testing

Radio head monitoring is a long term test designed to exercise the radio head over a period of time. This allows installation crews time to adjust or modify the installation.

To monitor radio head operation:

  1. Select the TEST menu, and click on SELECT TEST. Choose the RRH option.
  2. Click on the MONITOR menu option (Optionally press CTRL + M). If using Classic menus, the text under the MONITOR button changes to a green ON.

  3. Click RUN (Optionally press F5.

    The word "Monitoring" will appear in yellow at the top right of the window to indicate status. The test results pane shows the current values for the test.

  4. To stop monitoring, select the Test menu and click on STOP
