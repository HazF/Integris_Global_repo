Introduction

Introduction

The Absolute Analysis Integris 2000 RAN Tester is designed for field installation teams deploying Radio Access Network (RAN) equipment such as Remote Radio Heads (RRH) and their associated cabling and fiber connectors.

The one-touch Radio test exercises the RRH via its cables and connectors and verifies that each component of the RAN is functioning and installed correctly. Problems associated with configuration, component selection, faulty components, interoperability and installation will be detected.

If problems are encountered additional tests are available to isolate and fix issues.

Selecting Classic & Modern Menus

Selecting Classic & Modern Menus

Menu Location  
GUI or Keyboard None
Classic menu System > Prefs...
Modern menu System > Preferences...

Integris allows you to select between two menu structures to facilitate easier operation depending on the working environment.

  • Classic menus are better when using a touch screen
  • Modern menus are closer to standard Windows menus and may be easier to use with a keyboard and mouse.

In the sections that follow task descriptions will show the equivalent locations in both menu systems, e.g. "Classic menu Test > LoadConfig" means go to the Classic Test menu and select the Loadconfig menu item.

To swap between menu systems:

  1. Open the System menu
    • Click on Prefs on the Classic menu system, or
    • Click on Preferences on the Modern menu system
  2. Select either the Classic or Modern menu system as required
