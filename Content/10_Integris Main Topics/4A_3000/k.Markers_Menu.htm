The Markers Menu

The Integris 3000 has three optional markers that can be used to track various locations on the spectrum. A markers Table can also be enabled that gives additional information about points in the trace.

Classic Menu

Marker (1, 2, 3) Selects the current marker

Active Activates the currently selected marker

Freq (Soft Keypad) Sets the frequency for the current marker

Reference Sets the current marker as the reference for delta time and frequency measurements on the Markers Table.

Goto Peak Places the current marker at the highest amplitude peak value on the spectrum

Next Peak cycle the current marker between the peaks on the spectrum graph.

All Off Disables all markers

Table Enables/disables the onscreen Markers Table.

Markers Menu

Modern Menu

Configure Brings up a Markers configuration panel

All Off Disables all markers

Show Table Enables the onscreen Markers Table
