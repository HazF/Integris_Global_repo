Using Markers

The Integris 3000 has three optional markers that can be used to track various locations on the spectrum.

Menu Location  
Classic Markers Menu  
Modern Markers Menu  

Markers Configuration Panel

To set a marker:

  1. Open the Markers menu.
  2. Click on Configure. This will open a Markers configuration panel. (See Fig 1.1 )
  3. You now have a number of options:
    • Click on Active to activate the desired marker.
    • Type the desired frequency into the Frequency box. (Markers > Freq on classic menus)
    • Select the Ref box to make this the reference marker on the Marker Table.
    • Click on Go Peak to move the marker to the point of greatest Amplitude.
    • Click on Next Peak to cycle the marker through the peaks on the spectrum graph.
  4. Click on the All Off menu item to deactivate all the markers (though the Markers Control Panel remains on the screen).
  5. Click on Show Table to enable the onscreen Markers Table ("Using the Marker Table" on page 1).
