The Trace Menu

The Trace Menu

Classic Menu

Trace:

  • Realtime: Displays data for the current trace
  • Max: Shows the cumulative maximum value of each display point over many trace sweeps
  • Min: Shows the cumulative minimum value of each display point over many trace sweeps.
  • Average: Shows an exponential average of a number of traces

Enable Selects or deselects the current trace

Refresh Rate (0 - 1000) Sets the current refresh rate for the screen in milliseconds (ms)

Hold Time The period of time plotted values are held before the start to decay towards zero, in milliseconds (ms)

Decay Time The time taken for plotted values to fall to zero, in milliseconds (ms)

Reset Resets all values to the defaults

All Off Turns off the current trace

Traces Menu

Modern Menu

Configure Brings up a Trace configuration panel

  • Realtime: Displays data for the current trace
  • Max: Shows the cumulative maximum value of each display point over many trace sweeps
  • Min: Shows the cumulative minimum value of each display point over many trace sweeps.
  • Average: Shows an exponential average of a number of trace sweeps.

Refresh Rate (0 - 1000) Sets the current refresh rate for the screen in milliseconds (ms)

All Off Turns off the current trace
