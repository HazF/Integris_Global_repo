Using the Marker Table

Selecting Table on the Markers menu displays a table of the three available spectrum Markers and their associated values (See Figure 1.1. ).

Menu Location  
Classic Menu > Markers > Table  
Modern Menu > Markers > Show Table  

To use the Markers Table:

  1. Enable the Table using Markers > Show Table (Markers > Table on classic menus)
  2. Set one marker as a reference by clicking to select it in the Ref box on the Marker Control Panel. (Markers > Reference on classic menus).
 
  • The Frequency and Power columns show the current values for this pointer.
  • The Δ Power column displays the difference between the power value for the reference marker (shown as a 0.000 db value) and the power value for this marker
  • The Δ Frequency displays the difference between the frequency of the reference marker (shown in the Δ Power column as a 0.000 db value) and the frequency for this marker
