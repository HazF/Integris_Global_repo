The System Menu

The System Menu

The System menu controls various options related to the overall operation of the system. The layout varies depending on whether you are viewing a live or saved spectrum.

Classic Menu

LoadConfig This option brings up a list of predefined configuration options related to different manufacturers and carriers.

Source (Options: Live, File) This option selects the spectrum source. Live data is collected from the input port, while File allows saved displays to be replayed.

File Name (Only available when Source=File) Clicking on this option brings up a list of saved spectra that can then be replayed.

Record Allows a Live spectrum to be recorded for replay later.

Snapshot Takes a screen shot and saves it to the Integris-3000\AA_SCREEN_CAPTURE\ folder.

Prefs Opens up the screen preferences menu.

You can choose to place the Control Panels at the center of the screen or to the right.

You can also swap between the Classic and Modern menu systems.

Reset Resets all menu values to default.

About Displays the software version number.

Exit Closes the Integris 3000 application.
