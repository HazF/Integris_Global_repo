Introduction

Integris Expert is part of the Integris suite of Radio Access Network (RAN) test tools that provides you with the ability to extract data from a duplex CPRI link.

CPRI link traffic consists of three data "Channels"

  1. I/Q Data that carries the various voice streams between radio heads and the Base Band Unit or Units (BBU)
  2. Two Command & Management channels, called the Fast C&M channel and the Slow C&M Channel. These are embedded Ethernet streams used for passing command information between radio heads and the base station.
  3. In-band signalling for passing link status and requests between network nodes

Integris Expert has two main functions:

  1. Extract Fast or Slow Command and Management (Fast or Slow C&M) channel data, and send it live to a previously installed Wireshark application. Wireshark is a cross-platform open-source packet analyzer issued under the terms of the GNU General Public License. It is widely used for network analysis and communications protocol development, and comes pre-installed on your Integris system.
  2. Save raw CPRI frame data to an external file. This data is saved in binary form, but converted to Text format when first opened in Integris Expert.

In addition to the above, Integris Expert also monitors the link status and provides SFP Vital Product Data (VPD) and metrics.
