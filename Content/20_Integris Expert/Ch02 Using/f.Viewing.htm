Viewing a Raw CPRI File

Once a raw CPRI file has been captured, you can view its contents from within the Expert application

Click on the View Capture File button (See Fig 1.1 ). An explorer window opens allowing you to select any previously captured files.

Note: Raw CPRI link data is saved initially to a binary file. Binary files have an .ICF file extension. (See Fig 1.2 )

When you select a .ICF file for viewing for the first time it is automatically converted to a plain text file with an .ICV extension and opened in Windows Notepad.

Converted (.ICV) text files can of course be copied for use in other applications for post-processing.

Select the file you want to view and click on Open.
