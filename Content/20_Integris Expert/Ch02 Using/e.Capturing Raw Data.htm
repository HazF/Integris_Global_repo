Capturing Raw CPRI Data

Integris Expert allows you to capture the raw CPRI frame contents to a text file.

To capture raw CPRI traffic:

  1. Click on the Raw CPRI Frame Capture tab (See Fig 1.1 )

    The raw capture screen

  2. Click on the CPRI Link Speed dropdown menu and select the required link speed
  3. Click on the Capture(Bytes) dropdown menu and select the number of Bytes in each frame you want to capture (See Fig 1.2 )

    Note: The number of Bytes contained in a CPRI frame varies with the link speed.

  4. Click on the required Capture file size radio button, or select Custom and type in the file size you want. If selecting a custom file size, select either Mega-Bytes (MB) or Giga-Bytes (GB) units from the drop-down menu.

    The Capture(Bytes) dropdown menu

  5. Type (or paste) a file name and path into the Output File pane. Alternatively use the Browse button to the right of the pane to use a Windows Explorer window to locate the required folder, and type a file name.

    Note: Raw CPRI link data is saved initially to a binary file with an .ICF file extension. When you want to view the file later it is automatically converted to a plain text file with an .ICV extension.

  6. When you have completed the above settings, click the On button. This will start capture to the designated file. (See Fig 1.3 )

    The Raw Capture buttons

  7. File capture will automatically stop when the file size reaches the value you set in step 4 above. If you want to stop capture before the set file size is reached, click on the off button.
