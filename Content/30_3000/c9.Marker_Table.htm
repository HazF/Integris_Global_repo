Using the Marker Table

Menu Location  
Classic Menu > Markers > Table  
Modern Menu > Markers > Show Table  

When enabled, the Markers Table is located below the on-screen graph(s) and displays amplitude and frequency offset values for each of the five markers.

Markers & the Marker Table

To enable the Markers Table using the Modern Menu system:

  1. Open the Markers menu and click Show Table. The Markers Table opens at the bottom of the screen under the graph(s)
  2. On the Markers menu, click on Configure. The Markers control panel opens (See Fig 1.1 )

    The Markers control panel

  3. Set one marker as a reference by clicking to select it in the Ref column.

To enable the Markers Table using the Classic Menu system:

  1. Open the Markers menu and click on Show Table. The Markers Table opens at the bottom of the screen under the graph(s).
  2. Set one marker as a reference by clicking on Reference and selecting the desired marker number.
 
  • The Frequency and Power columns show the current values for this pointer.
  • The Δ Power column displays the difference between the power value for the reference marker (shown as a 0.000 db value) and the power value for this marker
  • The Δ Frequency displays the difference between the frequency offset of the reference marker (shown in the Δ Power column as a 0.000 db value) and the frequency offset for this marker
