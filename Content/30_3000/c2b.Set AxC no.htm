Setting the AxC#

You can set which Antenna System Container (AxC) to monitor from the Radio dropdown on the CPRI menu.

AxC / Radio Number

Menu Location
Classic Menu CPRI > Radio
Modern Menu CPRI > Radio

To set the AxC / Radio number:

  1. Go to the CPRI menu on either the Classic or Modern menu systems.
  2. Click on the Radio menu item.
  3. Select the desired AxC / Radio number from the list.
