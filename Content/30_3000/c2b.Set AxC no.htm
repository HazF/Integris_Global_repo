Setting the AxC#

AxC / Radio Number

Location    
GUI or Keyboard None  
Classic menu Source > AxC
Modern menu Source > AxC

You can set which Antenna System Container (AxC) to monitor from the AxC dropdown on the Source menu.

To set the AxC / Radio number:

  1. Go to the CPRI menu on either the Classic or Modern menu systems.
  2. Click on the Radio menu item.
  3. Select the desired AxC / Radio number from the list.

Information: The number of available CPRI AxC containers in a frame vary depending on the selected link speed.
