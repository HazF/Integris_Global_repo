Using the System Status Pane

The System Status pane displays quick reference values for the test port status, and how key parameters have been set.

If the input source is Live (i.e. not from a recorded graph) it additionally shows if valid traffic is present on the input port. The Link state is described as “Down”, “Active” (See Fig 1.2 ), or if the SFP fitted in the Test port has become faulty then "Bad SFP" is displayed in red.

System Status Pane (Source=LIVE)

The panel is 'active' so values can be set by clicking on the parameter and setting a value using a keypad, a slider, or selecting the value from a list.

Clicking on the Link icon displays SFP information (See Fig 1.3 ).
