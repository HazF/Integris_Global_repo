Switching Between Menu Types

The Integris 3000 can be configured to use either the Classic Integris menu or a 'Modern' menu system. You can change between the two types using the Preferences menu item. This is located at:

  • System > Prefs using the Classic menu system (See Figure 1.1. )
  • System > Preferences on the modern menu system (See Figure 1.2. ).

The Classic type has larger menus and is more suited to touch screen operation. The Modern type is better suited to traditional mouse and keyboard operation.

The sections that follow use the Modern menu perspective by default, but the equivalent locations on both menu types are described.

Note: Some menu contents differ between Classic and Modern types
