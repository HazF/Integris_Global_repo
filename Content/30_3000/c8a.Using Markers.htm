Using Markers

Menu Location
Classic Menu Markers > Marker
Modern Menu Markers > Configure

The Integris 3000 has five optional markers that can be used to track frequency offsetss on the spectrum.

Note: Markers can only be used with Power Spectrum graphs, and when a single graph is active on the system.

To set a marker using the Modern Menu system:

  1. Open the Markers menu.
  2. Click on Configure. This will open a Markers configuration panel. (See Fig 1.1 )

    Markers Configuration Panel

  3. You now have a number of options:
    • Click the Active checkbox to activate the desired marker.
    • Type the desired frequency into the Frequency box.
    • Select the Ref box to make this the reference marker on the Marker Table.
    • Click on Go Peak to move the marker to the point of greatest Amplitude.
    • Click on Next Peak to cycle the marker through the peaks on the spectrum graph.
  4. Click on the All Off menu item to deactivate all the markers

  5. Click on the Show Table checkbox to enable the onscreen Markers Table

    (See "Using the Marker Table" on page 1).

    (See Using the Marker Table).

To set a marker using the Classic Menu system:

  1. Select the Markers menu.
  2. Click on Marker and select the desired marker number.

  3. The Active menu item will display "Off" to indicate that the selected marker number is not yet active. Click on Active to toggle it to "On". The magenta marker arrow will appear on the screen at frequency offset zero (0.0 MHz).

    Marker 1 enabled

You now have a number of options:

  • Click on the Freq menu item and type the desired marker frequency.
  • Click on the Reference menu item ands select the marker you wish to use as a reference on the Marker Table.
  • Click on Go Peak to move the marker to the point of greatest Amplitude.
  • Click on Next Peak to cycle the marker through the peaks on the spectrum graph.
  • Click on the All Off menu item to deactivate all the markers.
  • Click on Table to enable the onscreen Markers Table
