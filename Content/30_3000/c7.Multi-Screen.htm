Viewing Multiple Graphs

Menu Location  
Hot Keys CTRL + G  
Classic Menu Graph > New  
Modern Menu Graph > New  

You can view multiple live or recorded graphs on the same screen. This can be useful when comparing a live graph against saved examples, or inspecting different aspects of the same source.

Multiple Graphs & Control Panels

To add a new graph:

  1. On the Graph menu, select New, or press CTRL + G. The Create Graph dialog opens (See Fig 1.2 ).

    The Create Graph dialog

  2. Click on the Type dropdown and select the type of graph you want to display.
  3. Click on the Source dropdown and select either of the two live test ports (ports 1 and 2) or a previously saved file.
  4. Click on OK to add a new graph or Cancel to go back to the previous screen configuration.
  5. The original graph will now appear with blue positioning borders to allow you to place the new graph in relation to it. Fig 1.3 shows the left positioning bar selected which will place the new graph to the left of the current graph. Fig 1.4 shows the new graph displayed to the left of the original graph.

Selecting / Deselecting Multi Graph View

Selecting / Deselecting Multi Graph View

When multiple graphs are available on the screen a view control icon appears at top right of the GUI.

View a single graph (left) and all graphs (right)

When multiple graphs are being displayed, an icon representing a single graph is displayed (SeeFig 1.4 left, and Fig 1.5).

To view any of the individual graphs, click to highlight it, then click on the control icon(or double click the desired graph). The view control icon changes to that shown in Fig 1.5 right.

To return to multiple graph view, click on the view control icon (or double-click the onscreen graph). The view control icon changes to that shown in Fig 1.5 left.

Moving a Graph

Moving a Graph

You can move any of the graphs on the screen by right clicking on the graph and selecting Move from the menu that appears. Positioning bars will appear as in Fig 1.3 to allow you to choose a new postion for the selected graph.

Closing a Graph

Closing a Graph

You can close any a single graphs, or all graphs, as follows

To close a single graph, right click on the graph and selecting Remove from the menu that appears.

Removing a single graph

To close all graphs, open the Graph menu and click on Close All. A warning message will appear warning that the screen will now retun to the default graph (See Fig 1.8).
