Selecting the Graph Type

Menu Location
Classic Menu Graph > Type
Modern Menu Graph > Type

Integris 3000 allows you to select the graph type or a combination of graph types. Each type supports a different aspect of signal analysis.

Spectrum: The power magnitude of the input signal is displayed across the set frequency range

Waterfall: Spectrogram giving a visual representation of frequency in the X axis against time in the Y axis

Persistence: Power magnitude is plotted against frequency, with color denoting the percentage of time a plot point is placed at the current location

Selecting a graph type

To select the Graph type:

  1. Go to the Graph menu on either the Classic or Modern menu systems.
  2. Click on the Type menu item.
  3. Select the desired Graph type or combination.
