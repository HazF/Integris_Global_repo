The Amplitude Menu

Classic Menu

Full Scale Sets the currently displayed spectrum's amplitude to maximum

Calibrate Available for Live displays only (Source=Live). Uses an on-screen keypad and slider to set site specific power level calibration for the base of the displayed power spectrum.

Ref Level (on-screen keypad & slider) Sets site specific power level calibration for the top bar of the power spectrum..

Scale (on-screen keypad and slider) to set the vertical values for the power spectrum on-screen grid
