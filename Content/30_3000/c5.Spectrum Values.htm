Setting Spectrum Parameters

Menu Location
Classic Menu Trace > Configure
Modern Menu Traces> Trace

The Integris 3000 can display more than just the default real-time spectrum. Using the Trace Control Panel you can overlay or remove Maximum, Minimum, Real-Time and Average spectra.

To select a configuration on the Classic menu system:

  1. Open the Traces menu and click on the Trace menu item. This will open a selection of graph values as described above
  2. On the Traces menu click on the Enable to display the related values. You can open any combination, or all values on the same graph (See Fig 1.1 ).

Classic menu showing Real Time and Maximum values

To select a configuration using the Modern Trace Control Panel:

  1. Open the Trace menu and click on the Trace menu item. This will open the Trace Control Panel panel (See Fig 1.2 ).
  2. You now have a number of options:
    • Click on Enable to activate the desired spectrum type.
    • Click on Reset to clear and restart the spectrum.
    • Click on the respective Hold and Decay boxes to open a dropdown list that allows you to select the desired values. (Hold and Decay are only available for Maximum and Minimum graphs. Real-Time and Average are grayed out).
  3. Click on the All Off menu item to deactivate all Spectra (The Trace Control Panel remains on the screen).
