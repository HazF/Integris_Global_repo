Exporting a File for Debug

Menu Location  
Not available on the Classic Menu  
Modern Menu HELP > Debug > Capture Bit Stream  

Capture Bin Stream is a function that is only available on the Modern Help menu, and it allows you to capture 2 Megabytes of live raw data from the field.

When you click on this option It creates an AA_DEBUG_CAPTURE.icf file in the C:\Integris-3000\ AA_DEBUG_CAPTURE folder. This .ICF file can be replayed through the Integris Expert application using theView Capture File feature.

Exporting a debug file

Integris Expert can extract the three data "Channels" present in CPRI traffic from this file:

  1. I/Q Data carrying the various voice streams.
  2. Fast and Slow C&M Channels used for passing command information between radio heads and the base station.
  3. In-band signalling used to pass link status and requests between network nodes.
