Quick Setup for Integris 3000

This procedure describes a simple test sequence using a live source and the Classic menus:

Note: The Integris 3000 can be configured to use either the Classic Integris menu or a 'Modern' menu system. You can change between the two types using System Preferences. This is located at:

  • System > Prefs using the Classic menu system
  • System > Preferences on the modern menu system
  1. Review the Integris Safety and Compliance document included with your system for unpacking and starting the system.

  2. Follow the setup sequence to connect the Integris 3000 into the link to be tested.

    (See "Cabling for RF extraction", page 1 )

    (See: Cabling for RF extraction)

  3. If the application has not started automatically, double-click on the Integris 3000 icon on the Windows Desktop to start the Integris 3000 application. After a short period the Integris 3000 splash screen is displayed, followed by the Integris 3000 GUI.

  4. Select the Source menu from the menu selector at the bottom of the screen, then click the Link Rate button. Select the correct link rate or AUTO.

    When the correct link rate is set, the Link icon at the top of the Status Pane shows a green Active (See Fig 1.1 ).

    The Link Status icon

  5. Select the Source menu and click on Configure, then select one of the two test ports: Port 1 or Port 2.

    Note: You can select either of the two test ports or any recorded files you have previously captured and saved to the C:\Integris-3000\AA_IQS_TRACES folder. (See Fig 1.2 )

    The Link Status icon

  6. Do one of the following:

    1. On the Source menu, select Mapping and select the desired configuration file for the system under test.

      Or:

    2. click the Mapping button at the top center of the GUi and select the correct configuration file for the system under test.

      Or:

    3. click on the Auto Detect icon at the top center of the GUI. If a configuration file is available to dispay a valid graph, a thumbnail graphic showing the available graphs will appear allowing you to select the appropriate display.

    Auto Detect button

    Thumbnail graphic

    Information: Integris 3000 configuration files contain information on frequency, bandwidth and other information required to decode the spectrum from the CPRI stream.

  7. When you use Auto Detect to choose the Configuration File, Integris 3000 goes to the first AxC cell containing valid antenna data.

    If you wish to select a different AxC container then select the Source menu and click the AxC menu option. Then select the desired AxC container from the drop down list.

    Information: The number of available CPRI AxC containers in a frame vary depending on the selected link speed.
  8. On the Graph menu, click the Type button and select the desired graph type.

  9. On the Amplitude menu, click on the Full Scale button to center the spectrum. Use the Ref Level and Scale buttons and their sliders to manually fine tune the spectrum.

    The Amplitude menu

  10. On the Freq menu, click on the Full Span, then Center buttons to locate the spectrum on the screen. Click the Span button to manually set the position.

    The Frequency menu

    Now that you have established the graph on the screen, you can set a marker to show values of the highest amplitude peak on the graph.

  11. Go to the Markers menu and click on the Marker button. Select one of the five available markers.
  12. On the Marker menu, click on the Active button to activate the selected marker. The legend will change to ON. A small magenta marker will appear on the graph.
  13. On the Marker menu, click on the Goto Peak button. The Marker pointer will automatically seek the greatest amplitude point on the display.

    Clicking on Next Peak moves the current marker to the next highest peak on the graph.

You can record traces to be replayed later, or loaded into Integris Expert for analysis. Recordings are saved in the proprietary .IQS format and are stored by default in the C:\Integris-3000\AA_IQS_TRACES folder.

To record a trace:

  1. Click on the circular red icon above the camera icon at bottom left of the screen. A Select Destination File dialog will open
  2. Click OK to start recording.
