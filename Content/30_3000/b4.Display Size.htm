Display Size

Menu Location  
Classic Menu - Available on the Modern Menu system only  
Modern Menu View > Layout  

You can change the resolution of the Modern menu system controls and interface. This is sometimes useful when operating the system remotely on a different resolution screen.

Change the resolution by opening the View menu and hovering over the Layout menu option. Select the desired value from the list

Note: Only the control boxes and Status Panel will change size. The Graph portion of the screen remains the same.
