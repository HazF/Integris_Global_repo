Starting Integris 3000

Start the Integris 3000 application by double-clicking on the Integris 3000 shortcut on the Windows desktop.

NOTE: After clicking on the Integris 3000 icon the application takes a few seconds to display the splash screen. If you click on the icon again during this period a warning opens telling you a that a second instance is already running.

This opens the Integris 3000 Graphical User Interface (GUI). When you first start your system you will see the default GUI with Classic menus, as shown in (See Fig 1.1 ). in Fig 1.1.

After the initial start up, the system will remember the last settings and will typically appear as in Fig 1.2. (See Fig 1.2 ).
