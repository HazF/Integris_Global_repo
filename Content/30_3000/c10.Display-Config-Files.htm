Saving and Loading Display Configurations

Menu Location  
Hot Key: CTRL + Shift + S / CTRL + Shift + L  
Classic Menu System > SaveConfig / LoadConfig  
Modern Menu System > Save Config ... / Load Config ...  

Integris 3000 allows you to save display settings so they can be loaded later. This can save time by loading potentially complex screen configurations like multiple graphs and control panes.

To save a display configuration file:

Open the System menu and click on SaveConfig. A dialog box opens allowing you to type a name for this configuration into the lower window, or select an already existing configuration to overwrite from the top of the window.

Click on OK to save the file.

The Save System Configuration dialog

To load a display configuration file:

Open the System menu and click on Load Config... (LoadConfig on Classic menus). A dialog box opens allowing you to select from previously saved configuration files (See Fig 1.2 ).

Click on OK to load the selected file.
