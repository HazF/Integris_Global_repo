Taking a Screen Snapshot

The Integris 3000 has a snapshot feature that allows you to take screen shots when a set of values or the display is showing something you would like to save for later.

Images are saved in JPEG (.jpg) format in the Integris-3000\AA_SCREEN_CAPTURE\ folder. File names start with an 'SC' followed by the date and then the time (e.g.SC20151125_084539_695.jpg)

Click on the left-hand camera icon to take a screen shot. The icon briefly displays a number '1' to indicate a screenshot has been taken.

Click on the right hand 'eye' icon to select the previously saved screen shot you want to display.
