The Graph Menu

Classic Menu

Classic Graph Menu

Type Selects the graph type or combination of types

  • Spectrum: The power magnitude of the input signal is displayed across the set frequency range
  • Waterfall: Spectrogram giving a visual representation of frequency in the X axis against time in the Y axis
  • Persistence: Power magnitude is plotted against frequency, with color denoting the percentage of time a plot point is placed at the current location

Grid Displays a grid on the screen. The scale is set on the Amplitude menu (Scale menu item).

Barrier Places ‘Barriers’ above and below the waveform. These values are based on the configuration file that has been loaded (e.g. 5MHz) and should ‘frame’ the waveform

Crosshair Displays a set of crosshairs centered on the current mouse pointer position. The X & Y values are displayed at the top right of the screen

Touch Turns the touchscreen features on or off. Touchscreen must be set to ON when you want to zoom to a specific area of the spectrum.

Unzoom Returns the display to full spectrum width.
