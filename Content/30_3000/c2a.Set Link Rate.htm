Setting the CPRI Link Rate

Location    
GUI or Keyboard None  
Classic menu Source > Link Rate
Modern menu Source > Link Rate

The CPRI menu gives access to controls for both the CPRI link rate and the AxC (Radio) cell number.

To set the link rate:

  1. Go to the Source menu on either the Classic or Modern menu systems.
  2. Click on the Link Rate menu item.

  3. Select the desired Link Rate from the dropdown list, or Auto to let the application automatically select the Link Rate.

Setting Link Rate

NOTE: Once the correct link rate has been set the Link state on the Status Pane shows as “Active”
