The Integris 3000 Display

The Integris 3000 Classic Menu is displayed by default and this is how the screen appears after a default reset. It consists of six main areas (See Figure 1. 1: )

  1. System Status Pane This displays quick reference values showing how key parameters have been set. If System > Source is set to Live it additionally shows the link status (Options: Red “Down”, Green “Active”).

    The System Status area is 'live' so values can be set by clicking on the number and setting a value using a keypad or slider, or selecting the value from a list.

    (See: The System Status Pane on page 1)
