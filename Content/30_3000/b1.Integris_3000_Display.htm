The Integris 3000 Display

The Integris 3000 Display

The Integris 3000 display consists of six main areas (See Figure 1. 1: )

  1. System Status This displays quick reference values showing how key parameters have been set. If System > Source is set to Live it additionally shows the link status (Options: Red “Down”, Green “OK”). The System Status area is 'live' so values can be set by clicking on the number and setting a value using a keypad or slider, or selecting the value from a list.

    (See: The System Status Pane on page 1)

    (See: Using the System Status Pane )

    The Integris 3000 Screen

  2. Record Trace & Snapshot: Record a trace file to be replayed later, or take and display screen shots.

    (See: Taking a Screen Snapshot on page 1)

    (See: Taking a Screen Snapshot )

  3. Menu Selector: Displayed when you select Classic View in System > Preferences, This selector is located at the bottom of the screen and allows the different Classic menus to be displayed to the right of the screen.

    (See: Integris 3000 Menus on page 1 )

    (See: Switching Between Menu Types )

  4. Power Spectrum: A traditional “Bart’s Head” displaying amplitude against frequency. Combinations of Power, Waterfall and Persistent graphs can be displayed by selecting them on Graph > Type. (See: "Setting up the Power Spectrum Graph" on page 1See: Setting up the Power Spectrum Graph

  5. Crosshairs: These can be enabled or disable on Graph > Crosshair. The crosshairs follow the mouse position and the X & Y values are displayed at position (6) (see below).

    (See: "Using the Crosshairs" on page 1 )

    (See: Using the Crosshairs)

  6. Crosshair Values: These show the X & Y values at the mouse / crosshair position (5) (see above).

  7. Menus: The screenshot shows Classic Menus. Alternatively you can use standard Windows style Modern Menus. You can swap between the two menu types using System > Preferences.
