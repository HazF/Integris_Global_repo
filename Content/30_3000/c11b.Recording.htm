Recording & Replaying RF Data

Menu Location  
GUI: Record icon at bottom left of the screen  
Classic Menu Source > Record / Playback  
Modern Menu Source > Record / Playback  

You can record graphs to be replayed later,

  • Recording are saved in the proprietary .IQS format and are saved to the C:\Integris-3000\AA_IQS_TRACES folder.
  • Recordings are fixed duration of approximately 50 seconds, and have a file size of approximately 130 Megabytes.
  • File names start with the date (in YearMonthDay format), an underscore then the time (e.g. 20151125_084539_695.iqs)

Record Control

To record a graph:

  1. Either
    • Click on the circular red icon above the camera icon at bottom left of the screen (See Fig 1.1 ). A Select Destination File dialog will open to name and save the recording
    • Open the Source menu and click on Record. A Select Destination File dialog will open to name and save the recording
  2. Click OK to start recording.

To replay a graph:

  1. Open the Source menu and click on Playback. A Select File dialog will open to allow you to select a file for replaying
  2. Click OK to playback the file.
