Recording RF Data

You can record traces to be replayed later, or loaded into Integris Expert for analysis.

Note: This feature is different to Capturing Bit Stream Help > Debug.

(See "Exporting a File for Debug" on page 1)

(See: Exporting a File for Debug)

  • Recording are saved in the proprietary .IQS format and are saved to the C:\Integris-3000\AA_IQS_TRACES folder.
  • Recordings are fixed length with a file size of approximately 130 Megabytes.
  • File names start with the date (in YearMonthDay format), an underscore then the time (e.g. 20151125_084539_695.iqs)

To record a trace:

  1. Click on the circular red icon above the camera icon at bottom left of the screen. A Select Destination File dialog will open
  2. Click OK to start recording.
