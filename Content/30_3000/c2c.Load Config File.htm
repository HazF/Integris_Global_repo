Loading a Configuration File

Location    
GUI or Keyboard Configurationicon at the top of the Main Window, or Auto Detect  
Classic menu Source > Mapping
Modern menu Source > Mapping

In order to extract RF data from a CPRI data stream, information for the specific manufacturer's equipment is required. Integris 3000 configuration files list various parameters used by the application to decode and display the RF data.

Information: Integris 3000 configuration files contain information on frequency, bandwidth and other information required to decode the spectrum from the CPRI stream.

To load a configuration file do one of the following:

  1. On the Source menu, select Mapping and select the desired configuration file for the system under test.

    Or:

  2. click the Mapping button at the top center of the GUi and select the correct configuration file for the system under test.

    Or:

  3. click on the Auto Detect icon at the top center of the GUI. If a configuration file is available to dispay a valid graph, a thumbnail graphic showing the available graphs will appear allowing you to select the appropriate display (See Fig 1.1 and Fig 1.2 ).

Auto Detect button

Thumbnail graphic

NOTE: The correct Link Speed must be set before Auto Detect may be used. If the Auto Detect icon is not displayed at the top of the screen, then check that the Link Rate setting matches that of the link Integris 3000 is connected to.

When you use Auto Detect to choose the Configuration File, Integris 3000 goes to the first AxC cell containing valid antenna data.

If you wish to select a different AxC container then select the Source menu and click the AxC menu option. Then select the desired AxC container from the drop down list.

If Integris 3000 is unable to match the incoming data to an installed configuration file, it will display a "CPRI analysis was unsuccessful" warning.

Click on OK to close the warning. (See Fig 1.3 )

Information: From time to time you may need to update or add Configuration files. These are located in the C:\Integris-2000\Configs\ folder.
