The Integris 2000 Radio tests consist of a sequence of tests that initialize the link and Remote Radio Head (RRH). It then verifies that the link is capable of reliably supporting CPRI traffic at the current speed.

Note: Radio Head testing includes checking for a response from the radio head unit. If a simple loop-back is connected then the Radio Head tests will detect this and fail the test (shown as a red 'X').

If you want to self-test the Integris system with a loop-back connector then select the Selftest configuration file before testing.

 

Information: The Integris 2000 Radio tests can include enhanced inventory listing for some manufacturers' equipment. This can be useful for fault finding and verifying software and firmware levels.

In this group of tests, the results are expanded to include Radio Head inventory information, including Manufacturer, model and serial number, as well as firmware level and service data.

To run a Radio test sequence:

  1. Open the TEST menu and select SELECT TEST, then click on the RADIO option.
  2. Now click on RUN from the same menu (or press the F5 key),

Once the test sequence has been completed you can optionally save the test results by adding them to a report. To add to an existing or new report file, click on the yellow Add To Report button.(See 'Working with Test Result Files').

You can now choose a name for these results, or you can accept the default name based on the date and time of the test. Once you have finished editing the name, click on OK.

NOTE: The SFP results section lists information related to the SFP fitted to the Radio Head. These results should NOT be confused with those gathered during Integris SFP Tests. See "Testing an SFP", page 1.
