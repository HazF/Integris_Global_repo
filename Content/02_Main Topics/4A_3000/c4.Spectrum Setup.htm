Setting up the Power Spectrum Graph

The way the spectrum graph is set up depends on which aspects of the signal you want to observe. Integris 3000 monitors 100% of the signal allowing you to zoom in to inspect a smaller frequency band or shift the graph down to see a spike.

There are controls to vary parameters related to signal Amplitude and Frequency.

Amplitude

Menu Location
Classic Menu Amplitude
Modern Menu Amplitude

The Amplitude menu has four menu items, three of which open control panels with sliders and edit boxes to set values. (See Figure 1.2. )

Full Scale' is used to return the graph to nominal values.

Modern Amplitude Menu

Use Calibrateto set the Y axis values (dBm). This has the effect of moving the base of the graph vertically on the screen and sets the base dBm of the spectrum graph.

Ref Level sets the position of the Y axis values on the screen. This has the effect of moving the base of the graph vertically on the screen.

Scale Sets the spacing of the Y axis values for the power spectrum on-screen grid. This has the effect of condensing or expanding the graph vertically on the screen.

Full Scale Sets the currently displayed spectrum's amplitude to maximum

Frequency

Menu Location
Classic Menu Frequency
Modern Menu Frequency

The Frequency menu has five menu items, three of which open control panels with sliders and edit boxes to set values. (See Figure 1.4. )

Modern Amplitude Menu

Use Center to set the center frequency for the currently displayed spectrum.

Span sets the frequency span of the currently displayed spectrum. It has the effect of expanding or contracting the width of the graph.

Full Span Returns the currently displayed spectrum span to maximum value

VBW Modifies the Video Bandwidth. Reducing the bandwidth ‘smooths’ the waveform and produces a thinner trace by decreasing the amount of noise. This is useful when identifying a weak interferer which would otherwise be lost in the noise.

RBW Allows you to balance the amount of detail against the scan speed of the signal. On a signal that takes time to settle, increasing the Resolution Bandwidth allows faster resolution.
