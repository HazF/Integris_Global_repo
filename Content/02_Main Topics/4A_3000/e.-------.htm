Trace Config Opens a trace configuration pane, that allows you to select graph type, hold and decay values.

Trace Configuration Pane

Marker Config Opens a markers configuration pane, that allows you to activate, send to peak or set a marker as the reference..

Markers Configuration Pane

Ref Level (on-screen keypad & slider) Sets site specific power level calibration for the top bar of the power spectrum.

Scale (on-screen keypad & slider) Sets the Y axis values for the power spectrum on-screen grid.

Center (on-screen keypad) Sets the display center frequency.

Span (on-screen keypad & slider) Sets the frequency range over which the instrument will sweep.

FullSpan Returns the display to the full range as determined in the configuration file.

VBW Modifies the Video Bandwidth. Reducing the bandwidth ‘smooths’ the waveform and produces a thinner trace by decreasing the amount of noise. This is useful when identifying a weak interferer which would otherwise be lost in the noise.

.All Controls Opens all 'floating' configuration panels and sliders, consisting of:

  • Trace Configuration pane
  • Markers Configuration pane
  • Reference Level slider
  • Scale slider
  • Center Frequency dialog
  • Span slider
  • VBW selector
  • CalLevel slider
