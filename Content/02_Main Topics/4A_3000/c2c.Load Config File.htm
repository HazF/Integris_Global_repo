Loading a Configuration File

Location    
GUI or Keyboard Configurationicon at the top of the Main Window, or Auto Detect
Classic menu System > LoadConfig
Modern menu System > Load Config

In order to extract RF data from a CPRI data stream, information for the specific manufacturer's equipment is required. Integris 3000 configuration files list various parameters used by the application to decode and display the RF data.

To load a configuration file:

  1. Click on the Configuration icon at top center of the main screen.
  2. Select the required configuration file from the list. (See Fig 1.1 )

Alternatively:

  1. Go to the System menu on either the Classic or Modern menu systems.
  2. Click on the LoadConfig (Load Config on Modern menus) menu item.
  3. Select the required configuration file from the list. (See Fig 1.1 )

To load a configuration file using Auto Detect

  1. Click on Auto Detectto the right of the Configuration icon to request the application to detect and load it for you.
  2. If Integris 3000 is unable to match the incoming data to an installed configuration file, it will display a "CPRI analysis was unsuccessful" warning. Click on OK to close the warning . (See Fig 1.2 )

See "Configuration Files", page 1 for more information on configuration files

NOTE: From time to time you may need to update or add Configuration files. These are located in the C:\Integris-2000\Configs\ folder.
