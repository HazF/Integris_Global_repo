Using the System Status Pane

The system Status pane displays quick reference values for the test port status, and how key parameters have been set.

If the input source is Live (i.e. not from a recording) it additionally shows if valid traffic is present on the input port (Options: Red “Down”, Green “Good”). If an SFP has become faulty then "Bad SFP" is displayed in red.
